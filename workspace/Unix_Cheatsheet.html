Command Cheatsheet
Command Description Usage
ls List working directory Displays the contents of the directory you're currently in.
tree Recursively list given directories via arguments. Displays directories in a recursive tree-esque fashion.
file Displays file type Recognizes and notifies you of the kind of data inside a file.
type - -
apropos - -
man Opens Unix manual Everything you need to know is in the manual. Use it.
cd Change working directory Use to navigate through directories in bash.
pwd Print working directory Displays your working directory in a list
mkdir Make directory Use to create a directory inside your working directory
echo - -
cat - -
less - -
